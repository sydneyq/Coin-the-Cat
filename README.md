# Coin the Cat Slack Bot

This is a Slack bot written in Python3. This project has been forked from the [Python slackclient library](https://github.com/slackapi/python-slackclient).

[![codecov][codecov-image]][codecov-url]

Requirements & Installation
---

This library requires Python 3.6 and above. 
> **Note:** You may need to use `python3` before your commands to ensure you use the correct Python path. e.g. `python3 --version`

```bash
$ python3 --version
```

> We recommend using [PyPI][pypi] to install the Slack Developer Kit for Python.

```bash
$ pip3 install slackclient
```

Need Help Starting?
---
Take a look at the [setup tutorial](https://github.com/slackapi/python-slackclient/blob/main/tutorial).
